# Tyler Stapler's Website

[![Cloudflare Pages](https://img.shields.io/badge/Cloudflare_Pages-Deployed-orange)](https://pages.cloudflare.com/)

This is my personal website ([live
version](https://tyler.staplerstaion.com)). It is deployed via **Cloudflare Pages** for global edge performance. It is built using the static
site generator [Hugo](https://gohugo.io) and the theme
[Espouse](https://github.com/tstapler).

This site uses
[HugoPhotoSwipe](https://github.com/GjjvdBurg/HugoPhotoSwipe/wiki) to
manage the photo gallery
